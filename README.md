# Object helper for PHP

A helper for PHP 7/8 to convert arrays and json strings to objects.

## Installation

You can install the package via composer:

```
$ composer require kfoobar/laravel-object
```

## Basic Usage

```
$object = Obj::make(['foo' => 'bar']);
```
